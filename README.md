# face detect model 

реализация собственной модели для предсказания ограничивающего прямоугольника лица (bbox).

Файлы:
- `model.py` — класс `FaceDetector` (без предобученных архитектур).
- `train.py` — скрипт обучения

ссылка на датасет: https://universe.roboflow.com/face-detection-dataset-g3kbp/face-detect-wqehm/dataset/4
